# Hey, I'm Alejandro

**Founder of [SquidRoster](https://squidroster.com)** — scheduling software that turns shift management into a team sport for hourly workers.

I'm a General Manager running food & retail operations at the WCS New York Aquarium, where I manage hourly and seasonal teams against labor percentage targets every week. I built SquidRoster because no scheduling tool could show me what a shift *costs* while I'm still building the schedule — so I made one that does.

**A decade of operating gave me the problem. A CS background gave me the ability to solve it.**

---

### 🦑 What I'm Building

**[SquidRoster](https://squidroster.com)** — Scheduling & Workforce Engagement SaaS for frontline teams

- AI-powered autofill with a 5-factor scoring engine
- Live labor-to-revenue KPI visibility *during* schedule building (not after)
- Duolingo-style gamification: tokens, badges, streaks, leaderboard, and a live Race Display
- Multi-tenant architecture · 47+ passing tests · React + TypeScript + FastAPI

Currently preparing for **Y Combinator Summer 2026**.

---

### 🛠 Tech Stack

**Frontend:** React 18 · TypeScript · Vite · React Query · Zustand · @dnd-kit  
**Backend:** Python · FastAPI · SQLAlchemy · PostgreSQL  
**AI:** MiniMax (text/chat) · GPT-4o-mini (vision) · AI schedule import (photo → shifts)
